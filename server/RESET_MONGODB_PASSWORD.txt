HOW TO RESET/CREATE MONGODB ATLAS PASSWORD
==========================================

OPTION 1: Reset Existing User Password
---------------------------------------
1. In MongoDB Atlas, go to "Database Access" in the left sidebar
2. Find your user (username: i222697)
3. Click the "Edit" button (pencil icon) next to the user
4. Click "Edit Password"
5. Enter a NEW password (remember this one!)
6. Click "Update User"
7. COPY the new password somewhere safe

OPTION 2: Create New Database User (Recommended)
--------------------------------------------------
1. In MongoDB Atlas, go to "Database Access" in the left sidebar
2. Click "Add New Database User" button
3. Authentication Method: "Password"
4. Username: Create a new one (e.g., "secure-messaging-user")
5. Password: Click "Autogenerate Secure Password" OR create your own
6. IMPORTANT: Click "Copy" to save the password - you won't see it again!
7. Database User Privileges: Select "Atlas admin" (for full access)
   OR "Read and write to any database" (more secure)
8. Click "Add User"

Then update your connection string:
mongodb+srv://NEW_USERNAME:NEW_PASSWORD@cluster0.g8dsdko.mongodb.net/secure-messaging?retryWrites=true&w=majority

IMPORTANT: Replace NEW_USERNAME and NEW_PASSWORD with your actual values!
